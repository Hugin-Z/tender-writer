# Contributing

欢迎提交 Issue 和 Pull Request。

## 代码规范

- Python 依赖见 `requirements.txt`,脚本风格参考 `scripts/` 现有代码
- 所有 Python 脚本通过 `run_script.bat` 在 venv 中执行

## 重要约束

**严禁在 PR 中提交任何真实招标文件、真实公司信息、真实项目数据。** 这是硬性红线。

示例和测试数据请使用虚构内容(参考 `examples/demo_project/`)。

## 想用真实项目试点?

请开 [Discussion](../../discussions) 而不是 Issue,我们可以私下沟通。
